# SvelteKit + Svelte 5 Runes + Tailwind — AI-Optimized Template

A clean and scalable template designed for AI-assisted development
(ChatGPT, Cursor, Copilot) using the latest Svelte 5 Runes.

---

## 🚀 Features

- SvelteKit (latest)
- Svelte 5 Runes
- Tailwind CSS
- AI-friendly folder structure
- Strict coding rules for consistent auto-generated code
- Aliased imports: $components, $api, $utils, $stores, $types

---

## 📦 Installation

npm install  
npm run dev -- --open
